# tark-test

### https://chrthemighty.github.io/tark-test/

### Installs all packages
```
yarn run install-packages
```

### Builds client
```
yarn run build-client
```

### Starts client
```
yarn run start-client
```

### Builds server
```
yarn run build-server
```

### Starts server
```
yarn run start-client
```
